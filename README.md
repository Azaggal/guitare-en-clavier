# guitare-en-clavier


## Matériels
### Pour ce TP, les éléments suivants ont été utilisés :
- Une guitare électrique
- Une carte Arduino
- Deux amplificateurs opérationnels LM741
- Sept résistances de 10kΩ et une résistance de 47kΩ
- Un oscilloscope
- Une alimentation stabilisée de 5V et une alimentation ±15V
